네이버 검색 어플
===========
- 네이버 검색 api를 활용한 앱
- 카페, 블로그, 사전, 이미지 검색 내용 확인 기능
</br>

## 주요기능
</br>

### 네이버 검색 기능
---
- 성인 단어 필터링
- 오타 필터링
- VIEW(블로그, 카페)의 클릭시 해당 웹 페이지로 이동 가능
- 스크롤을 통한 검색어의 추가 검색
- 검색 결과 중 최신 검색결과를 한글 내림차순으로 보여줌
</br>


### 최근 검색 단어 기능
---
- 최근 검색 단어 저장 및 삭제
- 최근 검색 단어를 통한 검색 기능
</br>


## 사용 기술 스텍
- **Android**
- **Kotlin**
- **MVVM**
- **ViewModel**
- **Retrofit**
- **LiveData**
- **open Api(naver)**
- **Glid**
