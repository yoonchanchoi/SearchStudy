네이버 검색 어플
===========
네이버 검색 어플로 View(블로그 + 카페), 사전, 이미지에 대한 검색 내용을 보여주며 통합페이지에서는 이미지를 제외한 내용을 합쳐서 보여줍니다.

## 사용 기술 스텍
**Android**, **Kotlin**, **MVVM**, **ViewModel**, **Retrofit**, **LiveData**, **open Api(naver)**, **Glid**
